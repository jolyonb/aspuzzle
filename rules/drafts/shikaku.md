# Shikaku Rules

Shikaku (also known as Rectangles or Divide by Box) is a puzzle played on a rectangular grid with numbered cells.

The objective is to divide the grid into rectangular regions according to the following rules.

## Rules

1. The grid must be completely divided into non-overlapping rectangles (or squares).
2. Each rectangle must contain exactly one number.
3. The number in each rectangle indicates the area of that rectangle (the total number of cells it contains).
4. Rectangles may not overlap and must cover every cell in the grid.

## Variations

* Shikaku can be played on non-rectangular grids.
* Some puzzles may have fewer given numbers, requiring more deduction.
