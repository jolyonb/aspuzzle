# Nonograms Rules

Nonograms (also known as Picross, Paint by Numbers, Griddlers, or Hanjie) is a picture logic puzzle played on a rectangular grid.

The objective is to shade cells to reveal a hidden picture according to the following rules.

## Rules

1. Each cell must be either shaded (black) or unshaded (white).
2. Numbers beside each row indicate the lengths of consecutive runs of shaded cells in that row, in order from left to right.
3. Numbers above each column indicate the lengths of consecutive runs of shaded cells in that column, in order from top to bottom.
4. Between each run of shaded cells, there must be at least one unshaded cell.
5. If a row or column has multiple numbers, the runs must appear in the given order.

## Variations

* Color nonograms use multiple colors, with separate clues for each color.
* Some nonograms reveal recognizable images when solved.
* Nonograms can be played on non-rectangular grids.
