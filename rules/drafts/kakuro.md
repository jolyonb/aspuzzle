# Kakuro Rules

Kakuro (also known as Cross Sums) is a number puzzle played on a grid resembling a crossword, with numerical clues instead of word clues.

The objective is to fill white cells with digits so that each run of cells sums to its clue.

## Rules

1. Each white cell must contain a digit from 1 to 9.
2. Black cells may contain clue numbers. A number in the upper-right portion indicates the sum for the cells to the right; a number in the lower-left indicates the sum for the cells below.
3. Each horizontal run of consecutive white cells must sum to the clue given to its left.
4. Each vertical run of consecutive white cells must sum to the clue given above it.
5. Within any single run (horizontal or vertical), no digit may be repeated.

## Variations

* Kakuro can be played on grids of various sizes.
