# Pipes Rules

Pipes (also known as Net or FreeNet) is a rotation puzzle played on a rectangular grid with pipe tiles.

The objective is to rotate the tiles so that all pipes connect into a single network.

## Rules

1. Each cell contains a tile with pipe segments pointing in various directions.
2. Tiles can be rotated (but not moved) to change which directions their pipes face.
3. All pipes must connect to form a single continuous network (no disjoint closed loops).
4. Every pipe endpoint must connect to another pipe.

## Variations

* "Wrap" variants allow pipes to connect across opposite edges of the grid.
* Pipes can be played on non-rectangular grids.
