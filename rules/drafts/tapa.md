# Tapa Rules

Tapa is a shading puzzle played on a rectangular grid with numbered clue cells.

The objective is to shade some cells to form a continuous wall according to the following rules.

## Rules

1. Some cells must be shaded black to form "the wall"; others remain white.
2. Clue cells contain one or more numbers and cannot be shaded.
3. Each number in a clue cell indicates the length of a consecutive run of shaded cells among the eight cells surrounding that clue.
4. If a clue contains multiple numbers, each number represents a separate consecutive run, and there must be at least one white cell between each run.
5. The runs can appear in any order around the clue cell.
6. All shaded cells must be connected orthogonally in a single group.
7. No 2x2 area may be entirely shaded.

## Variations

* A clue of "0" means all surrounding cells are white.
* Tapa can be played on non-rectangular grids.
