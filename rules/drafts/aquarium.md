# Aquarium Rules

Aquarium is a puzzle played on a rectangular grid divided into irregularly shaped regions called "aquariums."

The objective is to fill some cells with water according to the following rules.

## Rules

1. Each cell is either filled with water or empty.
2. Within each aquarium (region), water settles to the bottom according to gravity - if a cell is filled, all cells below it in the same aquarium must also be filled.
3. Water level is uniform across each aquarium's full width - water cannot be at different heights in different columns of the same aquarium.
4. Numbers outside the grid indicate how many cells must be filled in each row and column.

## Variations

* Aquarium can be played on non-rectangular grids.
