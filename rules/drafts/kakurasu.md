# Kakurasu Rules

Kakurasu (also known as Index Sums) is a logic puzzle played on a rectangular grid.

The objective is to shade some cells black so that the weighted sums match the clues.

## Rules

1. Each cell must be either shaded (black) or unshaded (white).
2. Each cell has a positional value equal to its position in its row and column (1st cell = 1, 2nd cell = 2, etc.).
3. The sum of the positional values of all shaded cells in each row must equal the number shown to the right of that row.
4. The sum of the positional values of all shaded cells in each column must equal the number shown below that column.

## Variations

* Kakurasu can be played on non-square grids.
