# Futoshiki Rules

Futoshiki (Japanese for "not equal") is a number-placement puzzle played on a square grid with inequality constraints.

The objective is to fill the grid with numbers according to the following rules.

## Rules

1. Each cell must contain exactly one number from 1 to N (where N is the grid size).
2. Each row must contain all numbers from 1 to N exactly once.
3. Each column must contain all numbers from 1 to N exactly once.
4. Inequality signs (< or >) between adjacent cells must be satisfied - the cell on the smaller side must contain a smaller number than the cell on the larger side.

## Variations

* Futoshiki can be played on non-square grids.
