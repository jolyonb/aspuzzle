# Dominosa Rules

Dominosa (also known as Domino Solitaire) is a puzzle where a full set of dominoes has been laid out in a rectangular grid, but the borders between dominoes have been erased.

The objective is to reconstruct where each domino was placed.

## Rules

1. The grid shows numbers representing the pips on dominoes, but domino boundaries are hidden.
2. Each cell must be paired with exactly one orthogonally adjacent cell to form a domino.
3. A domino is identified by its two numbers (e.g., 0-3, 2-5, 4-4).
4. Each possible domino combination appears exactly once in the solution.
5. For a standard puzzle using dominoes 0-0 through N-N, the grid will contain all possible combinations: (0,0), (0,1), ..., (N,N).

## Variations

* Different domino sets can be used (double-six, double-nine, etc.).
* For dominoes numbered 0 through N, the grid size is (N+2) x (N+1). For example, double-six dominoes (0-6) use an 8x7 grid.
