# Slant Rules

Slant (also known as Gokigen Naname) is a puzzle played on a grid where diagonal lines must be drawn in every cell.

The objective is to fill each cell with a diagonal line (either \ or /) according to the following rules.

## Rules

1. Every cell must contain exactly one diagonal line, either from top-left to bottom-right (\) or from top-right to bottom-left (/).
2. Numbers appear at grid intersections (corners where four cells meet, or fewer at edges).
3. Each number indicates exactly how many diagonal lines meet at that intersection point.
4. The diagonal lines must not form any closed loops.

## Variations

* Slant can be played on non-rectangular grids.
