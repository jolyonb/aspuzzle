# Binairo Rules

Binairo (also known as Takuzu, Binary Puzzle, or Binero) is a logic puzzle played on a rectangular grid.

The objective is to fill every cell with either a black or white circle (or equivalently, 0 or 1) according to the following rules.

## Rules

1. Each cell must contain exactly one of two symbols (black/white or 0/1).
2. Each row and each column must contain an equal number of each symbol.
3. No more than two consecutive cells in a row or column may contain the same symbol.
4. Each row must be unique (no two rows may be identical).
5. Each column must be unique (no two columns may be identical).

## Variations

* Binairo can be played on non-square grids, as long as dimensions are even.
