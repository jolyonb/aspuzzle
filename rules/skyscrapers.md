# Skyscrapers Rules

Skyscrapers is a number-placement puzzle played on a square grid. The objective is to place "buildings" of different heights (represented by numbers) in each cell such that the visibility constraints from outside the grid are satisfied.

Each number represents the height of a building, and when looking into the grid from any direction, taller buildings block the view of shorter buildings behind them.

## Rules

1. Each cell must contain exactly one number from 1 to N (where N is the grid size).
2. Each row must contain all numbers from 1 to N exactly once.
3. Each column must contain all numbers from 1 to N exactly once.
4. Numbers around the edge of the grid indicate how many buildings are visible when looking into the grid from that direction.
5. A building is visible if no taller building stands in front of it in the line of sight.
6. Buildings of the same height as an obstructing building are still blocked from view.

## Variations

* Some puzzles may have fewer clues around the edges, requiring logical deduction to solve.
